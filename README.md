# garmin-g5

## G5 – Instrumento Eletrônico de Voo em HTML


## Teclas de comandos para testes

| Ação          | Teclas                      |
| ------------- |:---------------------------:|
| Speed         | `1` ou `2`                  |
| Altitude      | `3` ou `4`                  |
| Pitch         | `ArrowUp` ou `ArrowDown`    |
| Roll          | `ArrowLeft` ou `ArrowRight` |
| Yaw           | `q` ou `e`                  |
